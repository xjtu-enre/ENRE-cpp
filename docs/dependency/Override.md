# Overrides

## Supported pattern
```yaml
name: Overrides Declaration
```
These reference kids indicate when a method in one class overrides a virtual method in a base class.

### Syntax: 

#### Examples: 

- Example1
```cpp
class A{
    public: virtual void func();
}
class B : public A{
    public: void func();
}
```
